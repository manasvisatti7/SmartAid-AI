# SmartAid-AI

SmartAid-AI is a web app that connects people in need with available volunteers using smart matching.

## Features
- Match volunteers with needs
- Priority-based sorting
- Simple UI for requests

## Tech Stack
- Frontend: HTML, CSS
- Backend: FastAPI (Python)

## How to run
1. Install dependencies
2. Run FastAPI server:
   uvicorn main:app --reload
3. Open index.html
